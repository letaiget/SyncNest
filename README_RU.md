# SyncNest (RU)

Десктоп/мобильное приложение для работы с общими папками и файлами в локальной сети и через интернет с realtime-синхронизацией.

## Язык

- Русская версия: `README_RU.md` (этот файл)
- English version: [README_EN.md](./README_EN.md)

## Статус проекта

Текущий этап: формирование архитектуры, документации и базового каркаса репозитория.

Начальная версия: `v0.1.0`

## Цель MVP (этап Desktop)

- Платформы: `macOS (Swift)` и `Windows (C#)`
- Единая учетная запись пользователя на устройствах
- Подключение/отключение устройств из приложения
- Создание и ведение общих папок с проектами
- Работа с файлами и папками (включая переименование/перемещение)
- Realtime-синхронизация
- Автоматическая блокировка редактирования файла:
  - первое устройство открывает файл на редактирование;
  - остальные устройства видят файл только в read-only.

## Принятые решения MVP

- Доступ: только для авторизованных пользователей (без гостевых ссылок)
- Регистрация: логин + пароль + email + код подтверждения
- Авторизация: логин + пароль
- Режим работы: только online (без офлайна на старте)
- Конфигурация сети: ручная через UI
- Очередь операций: `FIFO`
- Управление операциями: запуск + отмена (без паузы/резюма)
- Уведомления: только внутри приложения
- Корзина: есть, с автоочисткой через 30 дней
- История версий файлов: не входит в MVP
- Логи: серверные, по каждой сети, без автоудаления; есть вкладка просмотра в клиенте
- Шифрование: TLS в канале синхронизации; без at-rest шифрования в MVP
- Хранилище файлов: на пользовательском хосте (один хост на сеть)
- Центральный сервер: координация, учетные записи, метаданные, логи
- БД MVP: `SQLite`
- Серверный стек: `Node.js + TypeScript`
